== Description ==

Cloth QRCode is a powerful WordPress plugin that allows you to generate QR codes and manage their behavior. With Cloth QRCode, you can create QR codes that redirect to specific URLs, set expiration dates, add URL parameters, and track scan statistics.

= Features =
* Generate QR codes for any URL.
* Redirect users to specific URLs when they scan the QR code.
* Set expiration dates for QR codes.
* Add query or path parameters to URLs.
* Track and analyze scan statistics.
* Customize QR code appearance.

= Use Cases =
* Marketing campaigns with time-limited offers.
* Event management with QR code tickets.
* Product information and tracking.
* Analytics and user behavior tracking.

== Installation ==

1. Upload the `cloth-qrcode` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Go to the Cloth QRCode menu in your WordPress admin to start creating QR codes.

== Frequently Asked Questions ==

= How do I create a QR code? =

1. Go to Cloth QRCode > Add New in your WordPress admin.
2. Enter the URL you want the QR code to redirect to.
3. Customize the QR code settings as needed.
4. Publish the QR code.

= Can I set an expiration date for a QR code? =

Yes, you can set an expiration date for each QR code. After the expiration date, users will be redirected to a fallback URL if one is provided.

= How do I add parameters to the URL? =

When creating or editing a QR code, you can specify parameters in the QR Code Parameters section. You can choose between query parameters (e.g., `?param1=value1&param2=value2`) and path parameters (e.g., `/param1/value1/param2/value2`).

= How do I track scan statistics? =

Each time a QR code is scanned, the plugin logs the event. You can view the scan statistics in the QR code's edit screen under the Scan Statistics section.
